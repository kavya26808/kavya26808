### 🤖 AI-Powered Tourist Chatbot
An intelligent chatbot that assists travelers with personalized recommendations for tourist attractions, beaches, restaurants, temples, museums, hotels, and more in **Visakhapatnam**. Supports natural language conversations, multilingual interaction, voice assistance, real-time weather updates, Google Maps navigation, budget-based trip planning, and transportation suggestions (APSRTC buses, walking, cabs).

**Key Features:** AI conversational assistant • Personalized recommendations • Google Maps navigation • Real-time weather • Budget-based planning • Multilingual + voice support • Real-time responses via Ollama + Phi-3

**Tech Stack:** `Python` `FastAPI` `Uvicorn` `Ollama` `Phi-3 LLM` `NLP` `Pandas` `RapidFuzz` `Google Maps API` `Web Speech API`

🔗 **GitHub:** [AI-Tourist-Chatbot](https://github.com/kavya26808/AI-Tourist-chatbot)
